# Requirements

- installed go
- installed git
- basic familiarity with git and github
- stable internet connection

# Workflow

- make changes in source code ([main.go file](./main.go))

- run tests locally (optional) using `./test.sh` command

- submit exercise using `./submit.sh` command

**Important**: Before running scripts on linux you need to give them execution permissions using `chmod u+x ./<script-name>.sh` command (example: `chmod u+x ./test.sh`).

**Note**: Above steps are for linux users, if you are on windows run scripts using `.\<script-name>.bat` command where `<script-name>` is a script filename (example: `.\test.bat`).
